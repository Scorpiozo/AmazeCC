# AmazeCC AI Contributor Guidelines

---

When generating UI:

Always:

* Reuse existing components
* Reuse existing design tokens
* Reuse existing spacing
* Reuse existing typography
* Reuse existing shadows

Never:

* Create new card styles
* Create new button styles
* Create new spacing scales
* Create new shadow systems
* Introduce new colors
* Ignore accessibility requirements

When uncertain:

Use an existing component.

Do not invent a new one.
